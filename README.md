# NowPlus for Windows
A Fluent fullscreen clock, written in `.NET 10`, `WinUI 3` and `C++`.

> [!IMPORTANT]
> This app will NOT support Chinese (which is provided in [WinTimeBoard Legacy](https://github.com/ShihaoShenDev/wintimeboard-legacy-archive)) at this time<!--, even though I'm a Chinese speaker-->.
> 
> Instead, we only provide English edition.
